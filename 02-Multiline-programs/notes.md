## Example

The following code prints three lines of text, each on a new line:

```javascript
console.log("I");
console.log("know");
console.log("JavaScript");
```

A line break can also be inserted using the \n symbol.

## Empty Line

console.log() / console.log("") prints empty line.

```javascript
console.log("I");
console.log("");
console.log("know");
console.log();
console.log("JavaScript");
```
